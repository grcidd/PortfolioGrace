MES PROJETS

LE QUAI ANTIQUE

Le Quai Antique est un site web vitrine développé pour un restaurant gastronomique fictif. Il met en valeur l’ambiance chic et la carte raffinée de l’établissement à travers une interface élégante et responsive.
Technologies utilisées : HTML, CSS (Flex/Grid), JavaScript, PHP, MySQL

ECORIDE

EcoRide est une application web pensée pour permettre la réservation de vélos électriques en ligne. Elle comprend un espace utilisateur, une carte interactive, et un tableau de bord pour les gestionnaires.
Technologies utilisées : HTML, Tailwind CSS, JavaScript, PHP, Symfony, PostgreSQL

SAAS GESTION FREELANCE - PLATFORME DE GESTION POUR INDÉPENDANTS (En cours...)

Ce projet de SaaS (Software as a Service) a pour objectif d'aider les freelances et travailleurs indépendants à gérer leurs missions, factures, paiements et clients via une interface moderne, fluide et sécurisée. Il intègre un dashboard personnalisé, une authentification sécurisée, et des outils de suivi de projet automatisé.
Outils no-code utilisés : Bubble, Airtable, Make (ex Integromat), Stripe, Notion API
